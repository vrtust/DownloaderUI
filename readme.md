# DownloaderUI

DownloaderUI is a powerful and easy-to-use download manager built with C# and Avalonia. It provides a graphical user interface (GUI) for the [Downloader](https://github.com/bezzad/Downloader) , helping you manage and accelerate your downloads with advanced features.
